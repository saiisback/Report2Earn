# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- **DAO Governance Contract**: Decentralized autonomous organization for community governance
  - Proposal creation with deposits
  - Voting mechanism (Yes/No/Abstain)
  - Quorum-based execution
  - Time-based voting periods
- Modular architecture with reusable packages
- Open source documentation and contribution guidelines
- GitHub Actions CI/CD workflows
- Security policy and reporting process
- Code of Conduct and Contributor Covenant
- Dependabot configuration for automated dependency updates
- Comprehensive test workflows

### Changed
- Restructured project into modular packages:
  - `apps/ai-verification-module`: Reusable Python package
  - `apps/smart-contracts`: Shared Algorand contracts (now includes DAO)
  - `apps/report2earn`: Main application
- Updated README with badges, open source information, and DAO documentation
- Enhanced smart contracts package with DAO interaction utilities

## [0.1.0] - 2025-01-XX

### Added
- Initial release
- AI verification system with 5 multi-model consensus
- Algorand smart contract integration
- Next.js frontend with wallet connectivity
- Content scraping for multiple platforms
- Dynamic reward calculation system
- Image processing and manipulation detection
- Web search integration for fact-checking
